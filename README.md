# TkinterBattleship
GUI of the game Battleship using Python Tkinter 

How to use [Windows PC] :
- Install Visual Studio Code
- Open Command Prompt
- Navigate to where you want to store the code
- Clone repository

# Goal
Utilize Python Tkinter and a module type file structure
